## ✨ Features

### 🔍 **Smart Scheme Finder**
- Filter schemes by Category (Farmer, Student, Women, Senior Citizen, Business, Health, Housing)
- Filter by Income Level, State, and Age Group
- Instant results with official application links

### 📄 **Document Checklist Generator**
- Printable checklists for 20+ government services
- Aadhaar, PAN, Passport, Voter ID, Driving License, etc.
- Never go unprepared to government offices again

### 📞 **Emergency Helpline Directory**
- 20+ critical helpline numbers
- Searchable database
- One-click calling on mobile devices

### 🗂️ **Category-Based Navigation**
- Quick access cards for different citizen categories
- Visual icons for easy identification
- Bilingual support (English + Hindi)

### 🏛️ **Official Ministry Portals**
- Direct links to 12+ major government portals
- PM Kisan, Ayushman Bharat, PMAY, Mudra, etc.
- All links verified to .gov.in domains

### 📱 **Fully Responsive Design**
- Works on Mobile, Tablet, and Desktop
- Optimized for 2G/3G networks
- Lightweight (< 100KB)

### 🌐 **Bilingual Interface**
- Instant English ↔ Hindi toggle
- Accessible to rural and urban citizens alike

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS Variables, Grid, Flexbox
- **Vanilla JavaScript (ES6+)** - No frameworks, pure JS
- **Google Fonts** - Tiro Devanagari Hindi, Mukta, Poppins
- **LocalStorage API** - For language preference
- **Print CSS** - For checklist printing

**No external dependencies** - Pure, lightweight code!
